Edit process

  

Edit process

  

On this tab, you can view and edit the state of the process for the given contact. You cannot, however, edit the process itself as in Site manager -> Tools -> On-line marketing -> Automation processes. The step the contact is currently in is highlighted.

 

A set of actions is available to you based on the permissions in the security settings of the process:

 

Move to next step

There are two possible behaviors for this button that you can encounter:

•If there is only one step the process can move forward to, then clicking the button moves it to the next step.

•In case there is more than one step the process can continue to, clicking the button offers you more options to choose from. Each option names the step the process can continue to.

Move to previous step

Moves the process back to the previous step.

Move to a specific step

Moves the process to a step of your choice. Unlike the Move to next step and Move to previous step buttons, Move to a specific step requires you to have the Move to specific step permission for the On-line marketing module.

Comment and move to a step

Available after clicking next to the Move to next step and Move to previous step buttons. Opens up a dialog box in which you can enter a comment that will be attached to the move. The comment is then visible in the Process history. Choose the step you want to move into using the drop-down list above the comment box.

Remove from process

Removes the contact from the process.

 

Process history offers a way to see the history of this instance of the process.

 

For more information on managing processes, refer to On-line marketing guide -> Marketing automation -> Managing processes.
