Managing device profiles

  

Managing device profiles

  

You can modify the following properties of an existing device profile:

 

Display name

Name of the device profile used in the interface.

Name

Name of the device profile used in code.

Preview width (in pixels)

Determines the width of the device's preview in pixels.

Preview height (in pixels)

Determines the height of the device's preview in pixels.

Enabled

Check this box to enable the device profile for use.

Icon

You can Edit () or Delete () an existing icon. To replace the icon, click Update ().

Devices

List of devices to which the profile applies. Click Add items to choose additional devices.

User agents

Allows you to specify the exact user agents to which the profile applies.

 

Enter one user agent per line in accordance with RFC2616.

Macro

Use Edit condition () to specify a macro rule condition that devices must fulfill to be included in the profile.

 

 

 

Tip

 

If the system does not find a match among the Devices, it searches the User agents and then the Macro. This means you can specify User agents for devices that you do not have access to in the 51Degrees.mobi Lite Data.
