Contact role list

  

Here you can see a list of contact roles available on the current site and you can also create new contact roles. These objects enable you to assign a particular contact with a role, e.g. a contact works in a company as a designer so they are assigned with the role Designer.

 

If the Allow global configuration setting in Site Manager -> Settings -> On-line marketing -> Contact management -> Global data & merging is checked for the current site, the Site drop-down list is visible enabling you to view current site contact roles, global contact roles or a mix of global and current site contact roles. However, if global configuration is not allowed for the current site, only contact roles created on the current site can be viewed.

 

If available, the Site drop-down list offers you the following options:

 

(global and this site)

Both global and current site contact roles are listed. The option does not allow to create a new contact role.

(global)

Only global contact roles are listed. The option allows to create a new global contact role by clicking the New contact role link.

current site

Only current site contact roles are listed. The option allows to create a new site contact role by clicking the New contact role link.

 

 

 

Please note

 

If you are accessing this page from Site Manager -> Tools -> Contact management, the Site drop-down list allows you to work with all available sites.

 

 

Individual contact roles can be Edited (), Deleted (), Exported ( -> ) or Cloned ( -> ). The displayed list can be sorted by contact role name and, optionally, site. After clicking the icon in the header of the Actions column, a context menu is displayed, offering options for export of listed data to various types of files.
