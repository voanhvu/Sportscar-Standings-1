Order steps

CMS Desk > E-commerce > Orders > New order:

Order steps

  

Order steps

  

 

 

Please note

 

The following step order corresponds to the system's default checkout process, as available (in the CMS Desk order section) after a new installation of the CMS system. If you decide to customize the process, this step order and content may not be up-to-date.

 

 

Step 1

 

In this step you need to add an existing customer or create a new one. If you choose the Select an existing customer option, the Select button allows you to open a selection dialog and add an already defined customer. If you choose the Create a new customer option, you need to enter properties of the new customer, as described in more detail in the New customer topic.

 

Step 2

 

In this step you can see a list of all products added to the shopping cart, the total price and other order details. New products can be added by clicking the Add item link. This opens a new dialog allowing you to choose an order item, its quantity and, if defined, item options.

 

You can change the currency using the Currency drop-down list or enter a discount coupon code if available. By clicking the Empty button, the listed items can be removed from the shopping cart all at once. You can also remove only selected items if you check the boxes next to the respective items and click the Update button.

 

Step 3

 

In this step you need to enter the billing address and, if the shipping address is different from the billing address, the shipping address.

 

Billing address

 

Billing address

Allows you to select the customer's billing address (if available) or you can create a new one.

Name (company or personal)

Here you can enter the customer's name. The customer can be either a company or natural person.

Address lines

Allows you to enter the customer's address lines, e.g. a house number and street name.

City

Allows you to enter the city name.

ZIP

Here you can enter the ZIP code of the city.

Country

Allows you to select the customer's country. If the country has any states, another drop-down list is displayed. More details can be found in the Countries topic.

Phone number

Can be used to enter the customer's phone number; an optional field.

 

Shipping address

 

My shipping address is different from the billing address

If checked, a new form is displayed allowing you to enter the customer's shipping address.

Shipping address

Allows you to select the customer's shipping address (if available) or you can create a new one.

Name (company or personal)

Here you can enter the name of the customer. The customer can be either a company or natural person.

Address lines

Allows you to enter the customer's address lines, e.g. a house number and street name.

City

Allows you to enter the city name.

ZIP

Here you can enter the ZIP code of the city.

Country

Allows you to select the customer's country. If the country has any states, another drop-down list is displayed. More details can be found in the Countries topic.

Phone number

Can be used to enter the customer's phone number; an optional field.

 

Step 4

 

In this step you need to specify the payment and shipping methods. If the shopping cart does not contain any items that need shipping, only the Payment drop-down list is displayed allowing you to select a payment method based on the Allow if no shipping is supplied setting. More details can be found in the Payment methods topic in the Managing your store -> Orders chapter of the E-commerce Guide.

 

Shipping

Here you can select which shipping option will be used to deliver this order. Please note that the drop-down list is not visible unless the shopping cart contains items that need shipping.

Payment

Allows you to specify a payment method that will be used for the order.

 

Step 5

 

In this step you can see the order preview and thus check all information related to the order. Besides, you can add an order note and, by checking the Send new order notification checkbox, decide whether a notification about the new order will be sent to specified e-mail addresses (typically to the customer and to the merchant).

 

Step 6

 

In the last step of the checkout process you can perform the payment. Considering the selected payment method, the appropriate fields are displayed. The payment is finished by clicking the Finish payment button. To skip the payment click the Skip payment button.

 

 

 

Please note

 

If the taxes are set to be applied based on the shipping address but no items that require shipping have been put in the shopping cart, these are calculated based on the billing address.

 

 

More resources concerning checkout process can be found in E-commerce Guide -> Configuring your store -> Checkout process.

Further information about orders can be found in E-commerce Guide -> Managing your store -> Orders.
