Design (Device)

  

This tab allows you to modify the content of the page template in the same way as on the Design tab in CMS Desk. The template uses the edited device layout, which allows you to view the web part zones (and other layout content) defined for the given device profile.

 

 

 

Important!

 

The page template shares the same web part content and configuration between its default page layout and all device layouts. If the template's various page layouts use web part zones with identical IDs, any modifications made on this tab also affect the output of the remaining page layouts.

 

 

You can add add (), remove (), configure () or relocate web parts. Individual zones may also be managed via their drop‑down menu that you can expand by right‑clicking the header of the given zone or using the Web part zone menu () action.

 

Web part toolbar

 

Depending on the preferences of the current user, the Design tab may also include a toolbar that provides a convenient way to add new web parts. To determine which web parts are offered in the toolbar, you can select an appropriate category from the drop‑down list. It is also possible to filter the items by entering the name of a web part into the search textbox (). Once you find the web part that you wish to add, you can simply drag it from the main section of the toolbar and drop it at the desired location in one of the template's zones.

 

The Settings () button allows you to disable the toolbar or change its position. You can remove existing web part instances from the page template by dragging them into the "trash bin" area () of the toolbar.
