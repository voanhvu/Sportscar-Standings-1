Security tab

CMS Site Manager > Tools > Automation processes > Processes > Steps tab:

Security tab

  

Security tab

  

On this tab, you can specify who can move objects that are currently in the step to a different step. Note that in steps offering different cases in which the process continues in, you can set permissions for each case individually. You can see two sections in the tab:

 

Roles which can move object to different step

 

In this section, you can specify the roles whose members are allowed to move objects in this step to a different step. The following options are available:

 

None

Members of no roles are explicitly allowed to move objects to a different step. However, you can allow this to individual users in the second section.

Only listed

Members of the roles in the list below are allowed to move objects to a different step, unless they are among the All except listed users in the second section

All except listed

Members of all roles are allowed to move objects to a different step, except for members of the roles in the list below and the All except listed users in the second section.

 

If you select Only listed or All except listed, use the Select site drop-down menu to determine whether you want to choose from site or global roles. Then use the Add roles button to add roles to the list, and Remove selected to remove selected roles from the list. By clicking the icon, you open a drop-down menu with the Remove all option.

 

Users which can move object to different step

 

In this section, you can specify users who are allowed to move objects in this step to a different step. The following options are available:

 

None

Members of no roles are explicitly allowed to move objects to a different step. However, you can allow this to individual users in the second section.

Only listed

Members of the roles in the list below are allowed to move objects to a different step, unless they are among the All except listed users in the second section

All except listed

Members of all roles are allowed to move objects to a different step, except for members of the roles in the list below and the All except listed users in the second section.

 

If you select Only listed or All except listed, use the Add users button to add users to the list, and Remove selected to remove selected users from the list. By clicking the icon, you open a drop-down menu with the Remove all option, which removes all users from the list.

 

To learn more about configuring step security and automation process permissions, refer to On-line marketing guide -> Marketing automation -> Working with process designer -> Configuring security.
