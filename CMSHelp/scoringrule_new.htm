Kentico CMS 7.0 Context Help

New/Edit rule

CMS Desk > On-line marketing > Scoring > Score edit:

New/Edit rule

  

New/Edit rule

  

Here you can either define a new scoring rule or you can edit the currently selected one. Fill in the fields as required and click Save:

 

General

Display name

The name of the scoring rule used in Kentico CMS user interface.

Code name

The name of the scoring rule used in the code.

Value

The value added to a contact's score after scoring according to the current rule.

Rule type

The type of scoring rule. Choose either Attribute or Activity and set the rule settings as required.

•Attribute - indicates that the rule settings are based on an attribute.

•Activity - indicates that the rule settings are based on an activity.

 

Based on the selected rule type, different rule settings are available for attributes and activities.

 

Attribute

 

Rule settings

Attribute

Indicates on which attribute the current rule is based. Both built-in and custom attributes can be selected.

Condition

Allows to define a condition on which the scoring rule applies.

 

Activity

 

Rule settings

Activity

Indicates on which activity the current rule is based. Activities related to both built-in and custom file types can be selected.

Activity time

Indicates the time of activity logging. If the activity is logged within the defined time, the current rule is applied.

Activity value

Indicates what value the activity must have for the current rule to be applied.

Activity URL

Indicates what value the activity URL must have for the current rule to be applied.

Activity title

Indicates what value the activity title must have for the current rule to be applied.

IP address

Indicates what value the IP address from which a contact accesses the website must have for the current rule to be applied.

Activity comment

Indicates what value the activity comment must have for the current rule to be applied.

Validity

Recurring rule

Indicates if the rule is recurring. Using this setting, you can determine e.g. whether a contact should score on each page visit (TRUE) or only on their first visit (FALSE). The default value is FALSE.

Max. rule points

The most points a contact can score with the current rule.

Validity

Can be set either for a limited period of time (days, weeks, months, years, until a given date) or for unlimited period of time. To set your scoring rule validity for unlimited period of time, choose Until and leave the text box below empty.

 

 

 

Please note

 

The actual rule settings may vary based on the type of selected activity.
