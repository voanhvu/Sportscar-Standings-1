Kentico CMS 7.0 Context Help

Data.com

CMS Site Manager > Settings > Integration:

Data.com

  

Data.com

  

On this tab, you can configure the account which links Kentico CMS with Data.com and the mapping of contacts between the two systems.

 

General

API access token (required)

The token enables you to use the Data.com API interface. In order to obtain the token, you have to be a member of the Data.com developer program.

User login

This is your Data.com user account login.

User password

This is your Data.com user account password.

 

Mapping

Contact

Using the Edit button, you can configure the mapping of contact attributes between Kentico CMS and Data.com contacts.

Company

Using the Edit button, you can configure the mapping of company attributes between Kentico CMS and Data.com contacts.

 

Detailed information about Data.com integration in Kentico CMS can be found in Online Marketing Guide -> Integration of 3rd party modules -> Data.com.
