E-mails tab

  

On this tab, you can specify how will be notified about documents during the workflow cycle of a document.

 

The system sends notification e-mails according to the following rules:

 

•When a document reaches the step, the Waiting for approval e-mail is sent to users allowed to approve or reject a step.

•When someone approves a document in a step, i.e., moves to the next step, the Approved e-mail is sent to the user who submitted the document for approval.

•When someone rejects a document, i.e, moves it to the previous step, the Rejected e-mail is sent to the user who submitted the document for approval.

•When a document reaches the Published step, the Published e-mail is sent to the user who moved the document from the Edit step.

•When a document reaches the Archived step, the Archived e-mail is sent to the user who moved the document from the Edit step.

 

This tab offers two configuration sections:

 

General

 

In this section, you can explicitly enable or disable e-mail notifications for the workflow, or you can make the step inherit e-mail settings configured in Site Manager -> Settings -> Content management -> Content.

 

Specific e-mails

 

In this section, you can specify whether individual types of e-mails will be sent. The settings also enable you to choose a template for the e-mails or create a new one.

 

By default, all e-mail types have the default template assigned.

 

For each type of e-mail, there are three buttons that allow you to manage the template associated with the e-mail:

 

•Select - brings up the e-mail template selector, where you can click a template to select it.

•Edit - opens the editing dialog, where you can modify the currently selected template.

•New - enables you to write a new template from scratch

 

After you're done editing the e-mail settings, click Save to save the changes.

 

Send notification e-mail to listed users

 

In this section, you can set up sending e-mails to other users, who are not involved in the workflow process. Click Add users to select users that you want to notify about changes in the workflow cycle of documents.

 

You can also Select, Edit and create a new template used for the notifications.
